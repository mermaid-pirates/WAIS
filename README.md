> WAIS(Web Accessibility Improvment Service)는 모든 사람이 공평하게 웹을 이용할 수 있도록 돕는, 저시력 시각 장애인을 위한 웹 접근성 개선 서비스입니다.

## ❓ How to Start
```sh
# start web front server listen localhost:3000
cd frontend/
npm i
npm start

# start backend server listen localhost:4000
cd backend/
npm i
npm start
```
